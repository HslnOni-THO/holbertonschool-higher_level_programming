# 🐍 Python - Serialization Project

## 🚀 Description

Ce projet vous fait découvrir les concepts fondamentaux de **marshaling** et **serialization** en Python. Vous apprendrez à transformer des objets Python en formats stockables ou transmissibles (fichiers, réseau), puis à les reconstituer.

🎯 **Objectif** : Comprendre et mettre en pratique la sérialisation pour mieux gérer les échanges de données dans les applications modernes.

---

## 📚 Concepts Clés

- **Marshaling** : Transformer des objets en une forme standard (souvent binaire) pour communication entre systèmes.
- **Serialization** : Sauvegarder l'état d'un objet (dans un fichier ou via réseau) pour le recharger plus tard.
- **Désérialisation** : Opération inverse, qui recrée l'objet initial.

---

## 🧠 Ce que vous allez apprendre

- Différencier marshaling et serialization.
- Utiliser des modules Python comme `pickle`, `json`, et `xml.etree.ElementTree`.
- Convertir des structures de données vers JSON, XML, ou binaire.
- Lire/écrire des fichiers sérialisés.
- Appliquer la sérialisation dans des cas réels : bases de données, web, sockets, etc.

---

## 🛠 Outils Python Utilisés

- `pickle` : Sérialisation binaire native.
- `json` : Format texte léger, très utilisé en web.
- `xml.etree.ElementTree` : Manipulation XML simple.
- `socket` : Communication réseau.
- `csv` : Conversion CSV ➜ JSON.
