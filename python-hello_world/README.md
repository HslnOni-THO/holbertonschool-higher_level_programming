Python - Hello, World!
Bienvenue dans le projet Hello, World! en Python. Ce projet marque votre première étape dans l’apprentissage du langage Python et vous permet de découvrir sa syntaxe simple et puissante.

📚 Objectifs d’apprentissage
À l’issue de ce projet, vous saurez expliquer et appliquer :

Comment utiliser l’interpréteur Python

Comment afficher du texte et des variables avec print

Comment manipuler les chaînes de caractères (strings)

Ce que sont l’indexation et le slicing en Python

Quelle est la norme officielle de style Python (pycodestyle) et comment vérifier votre code

🚀 Démarrage rapide
Écrire votre premier programme “Hello, World!” :

Ouvrez votre éditeur de texte (vi, vim, emacs).

Écrivez le code suivant :

python
#!/usr/bin/python3
print("Hello, World!")
Enregistrez le fichier sous le nom hello.py.

Rendez-le exécutable :

bash
chmod +x hello.py
Exécutez-le :

bash
./hello.py
Vous devriez voir s’afficher :

text
Hello, World!
📝 Bonnes pratiques et style de code
Commentaires : Utilisez # pour documenter votre code.

Shebang : La première ligne doit être #!/usr/bin/python3 pour rendre le script exécutable sur Linux.

pycodestyle : Utilisez la commande pycodestyle pour vérifier que votre code respecte les standards de style Python.

Fichiers : Tous vos fichiers doivent se terminer par une nouvelle ligne et être exécutables.
